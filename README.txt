Scrabbly helps you as an assistant in playing scrabble.

Find out Bingos you can form with your letters
Find out possible words starting with your query
Get words that contains a query that you input
Check if a word is valid
Analyze the letters you have to find out if you have a bingo stem, etc.

To run, type 'python scrabbly.py' from inside the Scrabbly folder.
